What is it?

Photon is an extensible JavaScript framework for developing dynamic web sites and applications.

Declarative Data Binding

Fast & Responsive

Photon transparently uses advanced fragment caching and partial rendering techniques to deliver fast responsive UI's.

Resource Friendly

One of Photon's core goals is to consume less resources than other, similar frameworks. Photon is not only efficient with memory, it also does not leak memory, even on older less compliant browsers that some poor developers are forced to support :(
