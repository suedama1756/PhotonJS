What is it?

Photon is an extensible JavaScript framework for developing dynamic web sites and applications.

Motivation

Photon was initially developed after encountering significant issues with Knockout, most notably memory consumption and leak issues. During development, however, the two frameworks have diverged significantly, and will continue to move in quite different direction in future. Photon does NOT tie itself to any particular MVC pattern (such as MVVM), it provides a rich toolkit which can be leverage to enhance your development regardless...
