Light Speed Web Development

Photon is an extensible JavaScript framework for developing dynamic web sites and applications. It is released under the MIT Licence Checkout the features listed below, or jump straight to the examples for more information.

Declarative Data Binding

Easily bind DOM elements to data using a consistent, concise and readable syntax.

Rich Templating

Photon provides powerful data driven templating features, including flow control and external loading. There is even experimental support for 3rd party templating engines.

Extensible

Photon is a toolkit that can be leveraged in many different ways to suit your style of development, It is fully extensible and plays well with other libraries.

Fast & Responsive

Photon transparently uses advanced fragment caching and partial rendering techniques to deliver a fast, responsive user experience.

Resource Friendly

One of Photon's original design goals was to consume less resources than other, similar frameworks. Photon is not only efficient with memory, it also does not leak memory, even on older, less compliant browsers that some poor developers are still forced to support.
