Building PhotonJS
========


1.  Install [NodeJS](http://nodejs.org)

2.  Execute the folloing in a terminal:

mkdir PhotonJS
cd PhotonJS
git clone https://github.com/suedama1756/PhotonJS.git Lib
git clone https://github.com/suedama1756/PhotonJS.Build.git Build
cd Lib/build
sh ./build.sh
cd ../output
ls
